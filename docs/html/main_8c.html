Detailed Description

Main program body.

This file includes the main set of functions to initialize and run a control system using various components like buttons, LEDs, USART, and buzzer. It includes initialization of GPIOs, a main loop with a finite state machine (FSM) handling, and handlers for button debouncing and state changes based on user inputs.
