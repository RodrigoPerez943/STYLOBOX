◆ FSM_USART

enum FSM_USART

Enumerator for the USART finite state machine. This enumerator defines the different states that the USART finite state machine can be in. Each state represents a specific condition or step in the USART communication process.

Enumerator
WAIT_DATA 

Starting state. Also comes here when data has been send or read

SEND_DATA 

State to send data
