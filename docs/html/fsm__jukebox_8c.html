◆ _parse_message()

bool _parse_message ( char * p_message,
char * p_command,
char * p_param )

Parse the message received by the USART.

Given data received by the USART, this function parses the message and extracts the command and the parameter (if available).

‍1. Split the message by space using function strtok()

  1. If there's a token (command), copy it to the p_command variable. Otherwise, return false
  2. Extract the parameter (if available). To do so, get the next token using function strtok(). If there's a token, copy it to the p_param variable. Otherwise, copy an empty string to the p_param variable
  3. Return true indicating that the message has been parsed correctly
Parameters
p_messagePointer to the message received by the USART.
p_commandPointer to store the command extracted from the message.
p_paramPointer to store the parameter extracted from the message.
Returns
true if the message has been parsed correctly
false if the message has not been parsed correctly

Create a new jukebox FSM.

Creates a new jukebox FSM.This FSM plays melodies using a buzzer and communicates via USART with commands:play: Play the current melody;pause: Pause/resume the melody;stop: Stop the melody;playrestarts it;next: Play the next melody;speed: Change melody speed (requires parameter > 0.1);select: Select a melody by number;info`: Get current melody info.

Parameters
p_fsm_buttonPointer to the button FSM
on_off_press_time_msButton press time in milliseconds to turn the system ON or OFF
p_fsm_usartPointer to the USART FSM
p_fsm_buzzerPointer to the buzzer FSM.
next_song_press_time_msButton press time in milliseconds to change to the next song.
Returns
A pointer to the button FSM
